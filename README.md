# 똥 피하기

파이썬 `pygame`으로 만든 간단한 똥 피하기 게임입니다.  
키보드 **좌우 방향키**로 캐릭터를 움직여 **하늘에서 떨어지는 똥을 피하는** 방식으로 진행됩니다.  
시간이 지날수록 **똥의 개수(난이도)가 증가**하고, **점수도 계속 올라가는** 구조입니다.

---

## 1. 요구 사항

- **Python 3.x (⚠️ Python 3.14 미만 권장)**  
  *pygame이 Python 3.14에서는 안정적으로 설치되지 않을 수 있습니다.*
- 필요 모듈
  - pygame
  - opencv-python
  - numpy
  - pandas
---

## 2. 설치 방법

1. 이 프로젝트를 다운로드한 뒤, 압축을 풀어 폴더를 준비합니다.
2. 프로젝트 폴더에서 아래 명령으로 필요한 모듈을 설치합니다.

   ```bash
   pip install -r requirements.txt
   ```

3. 설치 불가 시

   ```bash
   py -m pip install 
   ```
   *오류 발생 시 경로 확인! (압축 풀며 폴더가 중복되진 않았는지)*

---

## 3. 게임 방법

- **조작 키**
  - **←, → (좌우 방향키)**  
    캐릭터를 왼쪽 / 오른쪽으로 이동하여 떨어지는 똥을 피합니다.

- **게임 진행 방식**
  - 화면 위쪽에서 **똥이 계속 떨어지며**, 캐릭터와 부딪히면 게임 오버입니다.
  - **시간이 지날수록**
    - 떨어지는 **똥의 개수**가 늘어나, 난이도가 점점 올라갑니다.
  - 살아남는 시간이 길어질수록 **점수가 계속 증가**합니다.

---

## 4. 실행 방법

1. 위의 방법대로 `requirements.txt`를 이용해 모듈 설치를 먼저 완료합니다.
2. 프로젝트 폴더에서 아래 명령을 실행합니다.

   ```bash
   python main.py
   ```

3. 게임 창이 뜨면 플레이를 시작하면 됩니다.
4. 게임 종료시 닉네임, 얻은 점수와 레벨이 표시됩니다.
