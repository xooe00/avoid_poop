# 똥 피하기

파이썬 코드로 제작된 장애물 피하기 게임입니다.\\
키보드 좌, 우 방향키로 캐릭터를 움직여 하늘에서 떨어지는 장애물을 피합니다.\\
시간이 지날수록 장애물의 개수(난이도)가 증가하고, 점수가 쌓이는 구조입니다.

---

## 1. 요구 사항

- **Python 3.x (⚠️ Python 3.14 미만 권장; pygame 안정성 이슈)**  
  *pygame이 Python 3.14에서는 안정적으로 설치되지 않을 수 있습니다.*
- 필요 모듈: pygame, opencv-python, numpy, pandas
  
---

## 2. 설치 방법

1. Github에서 압축파일을 다운로드한 뒤, 압축을 풀어 폴더를 준비합니다.
2. 프로젝트 폴더에서 아래 명령으로 필요한 모듈을 설치합니다.

   ```bash
   pip install -r requirements.txt
   ```

3. (위 방법으로 설치가 되지 않을 시) 아래의 명령을 한 줄씩 실행합니다.

   ```bash
   py -m pip install pygame==2.6.1
   py -m pip install opencv-python==4.12.0.88
   py -m pip install numpy==2.2.6
   py -m pip install pandas==2.3.3
   ```
   *오류 발생 시 경로 확인! (압축 풀며 폴더가 중복되진 않았는지)*

---

## 3. 게임 방법

1. 앞선 모듈 설치를 먼저 완료합니다.
2. 프로젝트 폴더에서 아래 명령을 실행합니다.

   ```bash
   python main.py
   ```

3. 게임 창이 뜨면 플레이를 시작하면 됩니다.\\
   + 게임 시작 → 닉네임(영어) 설정 → 난이도 선택\\
   + 조작 키: ←, → (키보드 좌,우 방향키)\\
   + 화면 위쪽에서 장애물이 계속 떨어지며, 캐릭터가 부딪히면 게임 오버\\
   + 살아남는 시간이 길어질수록 점수와 난이도가 계속 증가\\
5. 게임 종료시 닉네임, 얻은 점수와 레벨이 표시됩니다.
